Milestone Report

Accomplishments

We are currently implementing the core bubble dynamics simulation using a half-edge mesh data structure. Key accomplishments include:

  • Established the code framework based on HW3, established the code architecture and outline tasks (huge thanks to Carl)
  • for the bubble class:
    • Implemented bubble deformation via Laplacian smoothing dV = inv(M)*L*X and dX = V.
    • Implemented bubble volume calculation using the divergence theorem on triangular faces and volume preservation scaling X_new = k*X, where k = pow(V0/V, 1/3)
  • Path tracing integration with a custom BubbleBSDF

Preliminary Results

While we haven't yet implemented the full rendering pipeline, our current implementation:

  • Simulates bubble surface tension effects through Laplacian smoothing
  • Maintain volume conservation
  • calculate iridescence through our custom BSDF implementation
